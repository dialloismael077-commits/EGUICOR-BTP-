=== EGUICOR BTP - Site Web ===

Ce dossier contient :
- index.html  → le site web complet (responsive, mode sombre, menu mobile, bouton WhatsApp)
- IMG_8776.jpeg → logo de l'entreprise

--- Déploiement GitHub Pages ---
1. Créez un dépôt public sur GitHub (ex: eguicor-btp).
2. Ajoutez index.html et IMG_8776.jpeg dans le dépôt.
3. Allez dans Settings > Pages > choisissez la branche 'main' et / (root).
4. Votre site sera disponible à une adresse comme :
   https://votre-nom.github.io/eguicor-btp/

--- Déploiement Netlify ---
1. Créez un compte sur https://app.netlify.com/
2. Cliquez sur "Add new site > Deploy manually".
3. Glissez-déposez le dossier contenant index.html et IMG_8776.jpeg.
4. Votre site sera accessible à une adresse du style :
   https://eguicor-btp.netlify.app
